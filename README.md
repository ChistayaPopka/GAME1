# Flatline arena

## Описание проекта

**Flatline arena** — 2D-игра разработанная на Python с использованием библиотеки Pygame.

Игрок исследует мир, избегает противников, собирает игровые предметы и набирает очки. В игре реализованы механики: движение персонажа, взаимодействие с игровыми объектами, система здоровья и продвижение по миру.

---

## Возможности игры

* стартовое меню с настройками;
* управление персонажем (W, A, S, D);
* система здоровья;
* подсчёт очков;
* игровой таймер;
* враги, преследующие игрока;
* обработка столкновений;
* экран победы;
* экран поражения;
* пользовательский интерфейс с отображением здоровья, очков и времени.


---

## Используемые технологии

* Python 
* Pygame

## Установка

1. Установить Python.
2. Установить необходимые библиотеки:

```bash
pip install -r requirements.txt
```

3. Запустить игру:

```bash
python main.py
```

## Управление

* **W** - движение вверх;
* **A** - движение влево;
* **S** - движение вниз;
* **D** - движение вправо;
* **Space** - стрелять по направлению движения;
* **ЛКМ** - стрелять по направлению курсора.
